Правила игры "Одиннадцать"

🔹 1. Общие положения

  • Колода: 52 карты (стандартная, без джокеров).
  • Цель: Набрать больше очков, собирая карты по особым правилам.

🔹 2. Подготовка к игре

  1. Раздача карт: Каждый игрок получает по 4 карты. На стол выкладывается 4 карты.
  2. Остальная колода кладется в центр стола.

🔹 3. Ход игры

  • Сумма 11: Картой из руки можно забрать любое количество карт со стола, если их сумма равна 11.
  • Особые карты:
    • Валет (J) → берет любые карты, кроме дам и королей.
    • Дама (Q) → берет только другую даму.
    • Король (K) → берет только другого короля.
    • Туз (A) → считается как единица.
  • Если игрок не может взять карты, он обязан сбросить одну свою карту.

🔹 4. Счастливые карты

  • ♣2 (крестовая двойка) → +1 очко
  • ♦10 (бубновая десятка) → +1 очко

🔹 5. Окончание игры и подсчет очков

  • У кого больше карт → +2 очка
  • У кого больше карт масти ♣ (крести) → +1 очко
  • Наличие ♣2 → +1 очко
  • Наличие ♦10 → +1 очко
